# 📝 智能简历生成器 (Mac Native Resume Generator)

> **一款专为 Mac 平台深度打造的 100% 离线纯原生（PyQt6 架构）智能 A4 单页简历排版与 Word/PDF 一键生成软件。**

拒绝基于 Chromium 臃肿内核的 Electron “套壳网页”方案，本项目采用 **100% 原生桌面开发技术**（PyQt6 纯原生引擎），内存占用仅 **10MB 左右**，启动毫秒级响应。支持本地 103 套高档 A4 单页 Word 简历模板一键精准语义套用、AI 文本智能提取、像素级自适应微调，并内置 GitHub Release 版本自动检测更新机制，是一套真正产品化、开箱即用的专业简历利器。

---

## 🚀 核心亮点特性 (Features)

* **💻 极致流畅：macOS 100% 原生纯软件版**
  - 脱离任何网页 DOM 树与 WebView 壳，基于 macOS 底层渲染引擎与 PyQt6 架构独立运行。
  - 沉浸式暗色调高端 UI 设计，原生视网膜（Retina）窗口高保真缩放，完美适配现代 Mac 系统。

* **📄 万能 Word 经历克隆与格式缩紧 (docx XML 替换)**
  - 搭载底层 Python `docx` 语义填充大脑。不同于毁坏原文档样式的常规替换，本项目深入 **Word 的 XML 节点段落**，针对 `runs` 进行细粒度文本写入，百分之百保留模板原汁原味的高档字体、加粗及饱满黛蓝/竹青色底色。
  - **动态克隆/销毁经历**：当用户经历条数与模板不同时，系统会在底层自动进行 XML 节点的安全克隆或剪裁。
  - **智能段间距行高自动缩紧**，确保生成的 `.docx` 文件完美锁定在标准 A4 单页之内，提供极致排版体验。

* **🎨 1:1 A4 悬浮高真预览与像素级微调**
  - 内置基于 `QTextBrowser` 的富文本高拟真排版画布，保证所见即所得。
  - 集成**处女座级单页微调控制台**：支持“强制保持在一页内”的智能算法自适应字号收敛，同时提供像素级滑块，允许手动微调字号（11pt-16pt）与行距（1.2x-1.8x），告别跨页溢出。

* **💾 极速一键导出双端**
  - **PDF 导出**：调用 QTextDocument 的 `printToPdf` 原生接口，底层硬编码零边距及强制背景图渲染，一秒导出高清无瑕、无多余页脚页眉的 A4 简历 PDF。
  - **可编辑 Word 导出**：一键写入高精度的 `.docx` 文件，方便您后续在 Word 或 WPS 中进行任意二次手动改动。

* **🔄 独立软件级 GitHub 自动检查更新**
  - 启动时静默探测 & 提供右上角“检查更新”按钮。
  - 内置语义化版本号比对解析器（如比对 v1.0.0 与最新的 v1.1.0），若有新版，弹出漂亮的 macOS 对话框并一键原生拉起默认浏览器，直达您的 GitHub Release 下载页面！

## 🛠️ 技术栈盘点 (Tech Stack)

* **主框架**：Python 3.14 + PyQt6 (v6.11+) 原生桌面 GUI 引擎
* **数据流**：本地轻量级 JSON 结构化表单同步系统
* **Word 填充大脑**：基于 XML 树段落重组的 `python-docx` 保留样式高级渲染器
* **PDF 转换内核**：Qt 原生无损 `printToPdf` 物理级打印模块
* **打包体系**：PyInstaller 编译集成 + macOS 系统原生 `hdiutil` 磁盘映像 DMG 压缩技术

## 💻 开发者本地开发与打包指南 (For Developers)

如果您对本软件的源码感兴趣，或想定制自己的简历样式，可以按照如下步骤在本地部署开发环境：

### 1. 克隆仓库与配置环境
```bash
# 克隆本项目
git clone https://github.com/zyf2492313716-cloud/resume-generator.git
cd resume-generator

# 一键安装 Python 开发依赖库
pip3 install PyQt6 python-docx PyMuPDF pyinstaller --break-system-packages
```

### 2. 本地运行调试
```bash
python3 app.py
```

### 3. 一键编译打包 macOS 原生 DMG
本项目提供了一键打包脚本。您只需**双击运行项目根目录下的脚本**，系统将自动进行程序编译和磁盘映像压制，并在打包完成后自动在 Finder 中弹窗为您展示 `.dmg` 包！
* **项目内打包脚本**：`resume-generator/dist-desktop/mac-native/` 目录下
* **外层便捷入口**：直接双击 [打包Mac原生软件.command](../打包Mac原生软件.command) 即可！

## 📈 版本升级与检查更新配置闭环

项目内置的更新检查源已与您的 GitHub 仓库完美绑定。

当您开发了新功能，需要发布新版本时：
1. 本地双击 `打包Mac原生软件.command` 重新编译出最新的 `.dmg` 包。
2. 登录您的 GitHub 仓库，点击右侧的 `Releases` -> `Draft a new release`。
3. 创建版本标签为 **`v1.1.0`**（或任何高于 `v1.0.0` 的版本号），写好更新日志，并将打包好的 `智能简历生成器-原生版.dmg` 上传至 Release 附件框中发布。
4. 客户端软件在启动或手动点击 `🔄 检查更新` 时，将自动探知并引导用户下载，畅享专业独立软件的升级体验！
